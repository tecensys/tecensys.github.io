Relé de Enclavamiento

Este circuito muestra cómo hacer funcionar un relé de enclavamiento que permanece activado hasta que se corta el suministro de energía. En este punto, cuando se restablece la energía, el relé se desactiva. Este circuito se puede utilizar para activar una alerta audible y visible para una alarma, por ejemplo en puertas de emergencia.

En esta imagen, se muestra un circuito que incluye un diodo LED y un buzzer. Cuando el circuito es activado, funciona como una alarma audible y visible. La alarma solo puede ser desactivada si se corta la energía y se restablece.
